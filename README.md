# quick started 

 1. clone project
 ```
 ftp
 git clone https://github.com/dimoh4uk/site.git
 
 ssh
 git clone git@github.com:dimoh4uk/site.git
 ```
 
 2. go to new folder site 
 ```
 cd ./site
 ```
 
 3. install dependency
 ```
 npm i
 ```
 
 4. run gulp livereload server, the default browser should open  
 ```
 gulp
 ```
